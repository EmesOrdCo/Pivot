# PIVOT 1.0 Website

A beautifully enhanced recreation of the PIVOT 1.0 Shopify website with modern design, smooth animations, and improved user experience.

## Features

- **Modern Design**: Clean, minimalist aesthetic with enhanced visual hierarchy
- **Responsive Layout**: Fully responsive design that works on all devices
- **Smooth Animations**: Subtle animations and transitions for better user experience
- **Real Assets**: Uses actual images and video from the original Shopify site
- **Interactive Elements**: Mobile menu, search modal, and smooth scrolling
- **Performance Optimized**: Fast loading with optimized assets

## Features Included

1. **Hero Section**: Large, bold typography with animated background
2. **Video Section**: Embedded product video with elegant container
3. **Features Section**: Four feature cards showcasing:
   - Hidden Sharpener
   - Infinite Pencil
   - Detachable Ruler
   - Removable Eraser
4. **Newsletter Subscription**: Email signup form
5. **Footer**: Complete footer with payment methods and links

## Design Enhancements

- **Shadows**: Subtle depth with layered shadows
- **Gradients**: Modern gradient backgrounds for visual interest
- **Hover Effects**: Interactive elements with smooth hover states

## How to Use

Simply open `index.html` in a web browser. No build process or server required.

For local development with live reload, you can use:
- VS Code Live Server extension
- Python: `python -m http.server 8000`
- Node.js: `npx serve`
